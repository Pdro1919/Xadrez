Os Movimentos

♙ Peão → Avança apenas uma casa por movimento, apenas no primeiro movimento dele ele pode mover duas ♖ Torre → Essa peça pode andar em colunas, frente e para trás ♚ Rei → A peça mais importante do tabuleiro, onde consegue apenas se mover apenas uma casa para todas as direções
♘ Cavalo → Uma das peças mais legais para trolar aquele amigo, essa peça se move em L (2 casas para os lados + 1 para o outro) ♗ Bispo → Se move em diagonais e pode andar para trás ♕ Dama → Uma peça muito poderosa, capaz de imitar todos os movimentos já vistos (exceto o do Cavalo)
